They were using Rally in charge of all set up processes. My leader liked that I was successfully influencing others for good impact.

This feature unblocked several benchmark tools that require large disk space, and the machine has multiple small disks attached. ex.: Standard_D16ds_v6 has 2 local temp disks of 440 GiB, apply striping to get one of 880 GiB.
